# TaskManager

This repository contains frontend and backend for the TaskManager application, used to manage tasks and events. It's built using Java and Spring Boot with frontend in Angular.

# Stack
* **Language:** Java
* **Framework:** Spring Boot
* **Database:** PostgreSQL
* **Build tool:** Gradle
<br></br>
* **Frontend:** Angular
